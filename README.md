# BeautyGlow - Website Mỹ Phẩm Cao Cấp (Frontend)

## Giới thiệu

BeautyGlow là website mỹ phẩm cao cấp được phát triển bằng React và Tailwind CSS.  
Dự án tập trung vào giao diện người dùng hiện đại, trải nghiệm mượt mà và dễ mở rộng.  
Bao gồm hai phần chính:

- Trang người dùng: hiển thị sản phẩm, giỏ hàng, thanh toán, yêu thích, tài khoản, v.v.
- Trang quản trị: quản lý sản phẩm, tài khoản, khuyến mãi, đơn hàng và thống kê.

---

## Tính năng chính

### Người dùng (User)
- Xem sản phẩm theo danh mục
- Xem chi tiết sản phẩm, ảnh preview
- Thêm vào giỏ hàng, yêu thích
- Thanh toán qua VNPAY (giả lập)
- Đăng ký, đăng nhập, quản lý tài khoản
- Xem khuyến mãi, hướng dẫn, liên hệ

### Quản trị (Admin)
- Dashboard tổng quan
- Quản lý sản phẩm (thêm, sửa, xóa)
- Quản lý tài khoản người dùng
- Quản lý mã giảm giá
- Quản lý đơn hàng và trạng thái
- Thống kê doanh thu (đang phát triển)

---

## Công nghệ sử dụng

| Thành phần | Công nghệ |
|-------------|------------|
| Ngôn ngữ | JavaScript (React 18) |
| Framework | React + Vite |
| CSS | Tailwind CSS |
| Router | React Router DOM v6 |
| Thông báo UI | SweetAlert2 |
| Icon | Lucide React |
| Quản lý state | React Hooks (useState, useEffect) |
